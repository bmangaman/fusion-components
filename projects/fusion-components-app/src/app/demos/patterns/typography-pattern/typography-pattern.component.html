These type styles use LUCI as a basis, but for the most part have been tweaked to be smaller across the board. This better meets the need for our interfaces, which often need to display more condensed information.

Our applications use the Source Sans Pro font family. Four (4) font weights are provided to establish hierarchy, ensure legibility and create on-screen harmony.
